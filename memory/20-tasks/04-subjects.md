# Задача 4 — Темы писем (Subjects)

## Что реализовано
- `core/content.py` — спинтакс-парсер + макросы + `ContentManager.subjects`
- `gui/tab_content.py` — фрейм «Subjects» с превью и песочницей

## Спинтакс
Формат: `{word1|word2|word3}`. Вложенность поддерживается.

Алгоритм (`spin()`):
1. Regex `\{([^{}]+)\}` — ищет самый внутренний блок (без вложенных `{}`).
2. `random.choice(match.split("|"))` — выбирает случайный вариант.
3. Повторяет до тех пор, пока в строке есть `{...}`.

Пример:
```
{Привет|{Здравствуйте|Добрый день}}, {{name}}!
→ "Добрый день, John!"
```

## Макросы (плейсхолдеры)
| Макрос | Замена |
|---|---|
| `{{name}}` | Имя получателя |
| `{{email}}` | Email получателя |
| `{{senderName}}` | Имя отправителя |

Порядок обработки: сначала `spin()`, потом `substitute()`.
Нераспознанные макросы удаляются (regex `\{\{\w+\}\}`).

## GUI
- Кнопка «Load Subjects» + счётчик загруженных тем.
- Preview (CTkTextbox, read-only) — первые 5 raw-строк файла.
- Test Sandbox: 3 поля (Email, Name, Sender) + кнопка «Generate Preview».
- Вывод: 5 случайных сгенерированных тем (каждый клик даёт новые варианты).
